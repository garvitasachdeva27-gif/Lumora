# Lumora

Lumora is an AI powered adaptive learning platform designed to personalize the learning experience for every student. Instead of providing the same explanation to everyone, Lumora analyzes each learner's preferences, pace, strengths, and areas of improvement to deliver responses that are better suited to their individual learning style.

The platform combines conversational AI with learner profiling, progress tracking, and personalized recommendations to create a more engaging and effective learning experience.

Live demo at: https://lumorafrontend-chi.vercel.app/

## Features

- Personalized AI tutoring
- Adaptive learning profiles
- Context aware conversations
- Progress tracking
- Learning analytics
- Goal based recommendations
- Multi language support
- Secure user authentication

## Project Overview

Lumora was developed as a team project with the goal of making AI assisted learning more personalized and adaptive. Rather than functioning as a conventional chatbot, the platform focuses on understanding how a student learns over time and tailoring future interactions based on that understanding.

The system integrates conversational AI with learner profiling, study progress, and personalized recommendations to support a more effective and engaging learning experience.
